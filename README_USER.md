# CosyAudiobook (Mac版)

這是一個完全離線執行的 AI 有聲書產生器。

## 快速開始

1. **解壓縮**：將下載的 zip 檔案解壓縮到任何您喜歡的地方（建議放在「桌面」或「文件」）。
   
2. **啟動程式**：
   - 雙擊 `start_app.command`。
   - **首次執行會比較久（約 5-10 分鐘）**：因為程式會自動建立**兩個獨立的虛擬環境**（分別用於主程式與 CosyVoice3 模型），請耐心等待直到瀏覽器自動開啟。
   - 之後再次啟動只需幾秒鐘。

3. **使用**：
   - 瀏覽器會自動開啟 `http://localhost:8000`。
   - 上傳 EPUB 電子書。
   - 選擇喜歡的聲音（支援從 5 秒音檔複製聲音）。
   - 點擊「開始製作」。

## 注意事項

- **完全離線**：所有運算都在您的 Mac 上執行，不需網路，隱私絕對安全。
- **效能**：建議使用 M1/M2/M3/M4 系列晶片的 Mac。
- **首次使用模型**：第一次生成時會自動下載模型檔案（約 2-3GB），需保持網路連線。之後即可離線使用。

## 疑難排解

- 如果雙擊無法執行，請按右鍵選擇「打開」。
- 若看到「未識別的開發者」警告，請至「系統設定」>「隱私權與安全性」中允許執行。
